# RStudio quick run

Open the project at the repository root (or set your working directory to the repo root).

## 1) Tables
Edit and run:
- `rstudio/01_make_tables.R`

## 2) Simulation worked example (single scenario)
Edit and run:
- `rstudio/02_worked_example_sim.R`

## 3) MIMIC-IV demo implementation vignette (eTable 6)
Edit and run:
- `rstudio/03_worked_example_mimic_demo.R`

Notes:
- These wrappers call the corresponding scripts under `scripts/`.
- Outputs are written under `OUT_DIR` (created if needed).
